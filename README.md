# 沉浸式翻译 Lite (Immersive Translate Lite)

Read Frog 架构精简版 · Tampermonkey/Stay 用户脚本 · openai-compatible 自定义渠道 · tripleS 同人文专用

## 一键安装

**Stay (iOS Safari):**
1. Stay → + → 从URL添加
2. 粘贴: `https://raw.githubusercontent.com/KeyLiN913/immersive-translate-lite/main/immersive-translate.user.js`
3. 确认安装

**Tampermonkey (桌面):**
1. 浏览器打开上面的 raw 链接
2. Tampermonkey 自动识别 → 安装

## 功能

- 🆓 Google 翻译 (免费)
- 🐋 DeepSeek / 🤖 OpenAI / 🧠 Claude / 💎 Gemini
- 🔌 自定义 OpenAI 兼容渠道 (OpenRouter, Groq, SiliconFlow 等)
- 📝 双语对照 / 替换原文
- 📝 选中文本翻译
- 🎯 tripleS 同人文专用提示词
